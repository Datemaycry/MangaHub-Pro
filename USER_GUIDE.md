### ✨ Qu'est-ce que MangaHub Pro ?

MangaHub Pro est une application web qui vous permet de créer et de gérer votre propre bibliothèque de mangas numériques. Toutes vos données sont stockées **uniquement sur votre appareil**, garantissant une confidentialité totale et un accès hors-ligne.

---

### 📲 Installation de l'Application

Pour une expérience optimale, il est recommandé d'installer MangaHub Pro sur votre appareil. Cela lui permettra de fonctionner comme une application native, même sans connexion internet.

*   **Sur Ordinateur (Chrome, Edge) :**
    1.  Naviguez vers l'application dans votre navigateur.
    2.  Une icône d'installation (souvent un écran avec une flèche vers le bas) devrait apparaître dans la barre d'adresse.
    3.  Cliquez dessus et suivez les instructions pour installer l'application.

*   **Sur Android (Chrome) :**
    1.  Ouvrez le menu de Chrome (les trois points en haut à droite).
    2.  Sélectionnez **"Installer l'application"** ou **"Ajouter à l'écran d'accueil"**.

*   **Sur iPhone / iPad (Safari) :**
    1.  Appuyez sur le bouton **Partager** (l'icône carrée avec une flèche vers le haut) en bas de l'écran.
    2.  Faites défiler vers le bas et sélectionnez **"Sur l'écran d'accueil"**.
    3.  Confirmez en appuyant sur "Ajouter".

### 🚀 Comment Utiliser l'Application

#### ➕ Ajouter un Manga

Pour ajouter un nouveau manga à votre bibliothèque, cliquez sur le bouton **+** en haut à droite.

Vous avez deux options principales :

1.  **Import Rapide (Recommandé) :**
    *   Cliquez sur **"SÉLECTIONNER UNE ARCHIVE"**.
    *   Choisissez un fichier `.zip` ou `.cbz` sur votre appareil. L'application extraira automatiquement les images et tentera de récupérer les métadonnées si elles sont incluses.

2.  **Import Manuel :**
    *   Remplissez le titre, l'artiste, la série, etc.
    *   Sélectionnez les images pour les couvertures (début, fin, jaquette complète).
    *   Sélectionnez toutes les pages intérieures de votre manga.
    *   Cliquez sur **"Lancer l'import"**.

#### 🎓 Tutoriel de Bienvenue

La première fois que vous ouvrirez l'application avec une bibliothèque vide, un tutoriel interactif se lancera pour vous guider à travers les étapes de base : ajouter votre premier manga, l'inspecter et commencer la lecture.

#### 📚 Naviguer dans la Bibliothèque

*   **Recherche :** Utilisez la barre de recherche pour trouver rapidement un manga par titre, artiste, série ou tag. La recherche est "floue", elle pardonnera donc les petites fautes de frappe.
*   **Filtres et Tri :** Cliquez sur l'icône de filtre (entonnoir) pour :
    *   **Trier** votre bibliothèque par série, date d'ajout ou dernière lecture.
    *   Afficher uniquement les mangas avec un **marque-page**.
    *   **Filtrer** par tags.
*   **Sélection Multiple :** Cliquez sur l'icône de sélection (carré avec une coche) pour passer en mode sélection. Vous pourrez alors modifier ou supprimer plusieurs mangas à la fois.

#### 📖 Lire un Manga

*   **Ouvrir un manga :** Cliquez sur un livre dans l'étagère pour l'inspecter en 3D, puis cliquez sur **"Ouvrir"**.
*   **Ouvrir un chapitre :** Depuis la vue d'inspection, cliquez directement sur un numéro de page dans la liste des chapitres pour y accéder instantanément.
*   **Tourner les pages :**
    *   **Clic :** Cliquez sur les côtés gauche ou droit de l'écran.
    *   **Glisser :** Faites glisser une page avec votre doigt ou votre souris.
    *   **Clavier :** Utilisez les flèches `←` et `→`.
*   **Menu du lecteur :** Bougez votre souris ou touchez le milieu de l'écran pour faire apparaître les menus. Vous pourrez y placer un marque-page, changer le mode d'affichage (page simple/double), activer le mode nuit, etc.

#### 🎨 Personnalisation

Dans les **Paramètres Globaux** (icône d'engrenage en haut à droite), vous pouvez personnaliser de nombreux aspects de l'application :
*   **Thème Visuel :** Changez la couleur principale de l'interface.
*   **Étagères :** Modifiez le matériau et les gravures de vos étagères virtuelles.
*   **Animations & Sons :** Ajustez la vitesse des animations de page tournée, ou désactivez-les. Vous pouvez également régler le volume des effets sonores.

#### ⚙️ Gérer les Données

Dans les **Paramètres Globaux** (icône d'engrenage en haut à droite), vous pouvez :
*   **Exporter** votre bibliothèque (`.zip`). C'est une sauvegarde complète de tous vos mangas et de leurs métadonnées.
*   **Importer** une sauvegarde précédemment exportée.
*   **Purger** toutes les données pour repartir de zéro (attention, cette action est irréversible !).

### 🌐 Fonctionnement Hors-Ligne

Une fois l'application installée et vos mangas importés, **tout fonctionne sans connexion internet**. Votre bibliothèque est entièrement locale et privée.
